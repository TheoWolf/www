Cross-Platform

Datavyu is a Java-based application that runs on Windows and Mac operating systems. It requires no specialized hardware. Most labs can create Datavyu coding stations with computers already at hand. Datavyu builds on the OpenSHAPA coding tool.

Fully Extensible

Datavyu accommodates your needs. You decide how you want to code your video, audio, physiology, motion, or eye tracking data. Power users can write scripts in the Ruby programming language to extend Datavyu's functionality.

Open-Source

Datavyu software and technical support is free, now and always. Datavyu is open-source. Anyone can improve Datavyu and share the new features with other researchers.
